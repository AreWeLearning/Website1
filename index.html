Fundamental HTML Tags

Basic Tags

Tag Name Description
<!DOCTYPE html> 1st Item This tag shows the browser that the document type will be HTML. This is a self closing tag.
<head> Heading This is the tag that you would include <Link> tags for style sheets. You can also include <Title> tags to title the Tab in Google Chrome to your preference.
<Link> Link This tag links stylesheets.
<body> Website Body Most of the website's content should be posted in this section of the
code. From paragraphs to videos, this section contains it all!
<footer> Footer This part of the site is generally for navigation links to other parts of the site or can have a "back to top" button for long websites.
<a> Anchor It looks like <a href="https://www.wikipedia.org/">This Is A Link To Wikipedia </a>
